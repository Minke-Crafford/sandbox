# Sandbox

My personal sandbox for testing and building projects.

## Projects

This repository contains skeleton structures for multiple project types:

### 1. Go API (`go-api/`)
A REST API server built with Go's standard library.
- Basic HTTP server with multiple endpoints
- Handler structure for organizing routes
- Utility functions
- See [go-api/README.md](go-api/README.md) for details

### 2. Bash Scripts (`bash-scripts/`)
A collection of useful bash scripts and utilities.
- Example scripts (hello, system-info, backup)
- Reusable utility functions (logging, file operations)
- See [bash-scripts/README.md](bash-scripts/README.md) for details

### 3. Python Project (`python-project/`)
A Python package with proper structure and testing.
- Calculator and utility modules
- Comprehensive unit tests
- Example CLI application
- See [python-project/README.md](python-project/README.md) for details

## Getting Started

Each project has its own README with specific instructions. Navigate to the respective directory and follow the setup instructions.

## Structure

```
sandbox/
├── go-api/              # Go REST API project
├── bash-scripts/        # Bash scripts collection
└── python-project/      # Python package project
```
